# How to run app
## You need install docker
### docker-compose up --build in project folder

## You need such env variables
### PORT=5000
### DB_PORT=27017
### DB_HOST=mongo
### DB_NAME=VacationPay
### BASE_URL="http://localhost:3000" 
### ADMIN_EMAIL=den@mail.com
### ADMIN_PASSWORD=12345
### JWT_SECRET=SEcretKey
